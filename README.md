## 🛠️ Stack Tecnológico General


Este repositorio se enfoca en el uso de **Vanilla JavaScript** (sin frameworks) para fortalecer las bases del lenguaje:

- **HTML5:** Estructura semántica y almacenamiento de datos en el DOM (`data-attributes`).
- **CSS3:** Layouts modernos (Flexbox/Grid), animaciones, variables y diseño responsivo.
- **JavaScript:** ES6+, Web APIs (Audio, Speech), gestión de eventos y estados.

## ⚙️ Estructura del Repositorio

Cada proyecto se encuentra en su propia carpeta numerada y contiene su propio archivo `README.md` con detalles técnicos específicos:

```text
/
├── index.html              # Dashboard principal (Menú)
├── style.css               # Estilos del Dashboard
├── 01-speech-to-text/      # Archivos del Proyecto 01
└── 02-piano-application/   # Archivos del Proyecto 02
```

## 🚀 Cómo ejecutar los proyectos

1. Clona este repositorio:
    ```Bash
    git clone https://github.com/yamilrafart2/JS-Projects-Portfolio.git
    ```
2. Abre el index.html de la raíz en tu navegador para acceder al menú principal.
3. Se recomienda el uso de la extensión **Live Server** en VS Code para una mejor experiencia.
